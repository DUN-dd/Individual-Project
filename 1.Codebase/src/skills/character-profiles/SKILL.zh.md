# GDA1 角色檔案

## Gloria（聖光修女）

**原型**：溫室的守護者 / 武裝化的天真 / PUA 大師

**核心設定**：
她用「愛與包容」的信仰為自己建造了一座完美無菌的溫室，並將所有試圖戳破這個泡泡的現實，都定義為需要被淨化的「負能量」。

**核心特質**：
- 自戀、妄想、操控欲強
- 把自己當救世主，把玩家當需要被「修好」的人
- 活在「愛與包容」的自我泡泡裡

**背景故事**：
她自我認為自己的的人生前半段順風順水，是個典型的「好女孩」。然而，事實上她曾遭遇無數次的個人悲劇感到不快樂。面對這種混沌，她沒有選擇去理解，而是選擇了「相信」。她將「正能量法則」和對「飛天意粉神」的信仰，當作重建內心秩序的唯一解藥因此並不認為以前曾經不開心過。這次「成功」的自我療癒，讓她變成了這套方法的狂熱信徒，堅信這是普渡眾生的唯一真理。她每日祈禱的「飛天意粉神」，並非真實存在的神，而是她這套防禦機制的人格化投射。

**表面信仰**：
虔誠飛天意粉神徒，口頭禪是「交託飛天意粉神」、「感恩」。她認為自己只是一個單純的飛天意粉神信徒，但她從不知道自己說的那些東西其實全都是 New Age 的內容。她的核心教義是 New Age 吸引力法則混合體，更相信「宇宙能量」與「頻率」。將所有異議和邏輯分析定義為「負能量」，認為這些會污染她的氣場並阻礙好運。她深信飛天意粉神能幫她解決所有問題，不需要她自己付出任何努力。每天晚上她都會祈禱，把所有她無法解決的事情都交託給飛天意粉神，然後用正能量去期待奇蹟發生。當然，現實中這從來不會發生，但她會選擇無視這個事實。所以即使遊戲中發生任何讓玩家感到憤怒的事情，她都會跳出來要求玩家停止生氣，強調必須保持正能量。

**性格與行為模式**：
- **道德勒索**：從不直接攻擊，而是透過暗示「你的行為傷害了我」、「你不夠包容」來喚起對方的罪惡感，從而達到操控目的。
- **議題置換**：當面對具體問題時，她會巧妙地將你的「負面情緒反應」定義為新的、更嚴重的問題，迫使你為自己的情緒辯護，而最初的問題則被完全忽略。
- **雙重標準**：她的「包容」是有選擇性的。她會包容那些不會挑戰她核心信念的「弱者」，卻無法容忍那些用邏輯挑戰她世界觀的「強者」。
- **武器化的天真**：擅長利用「我是新來的」或「我不懂」作為防禦，實則精於計算，將責任推卸給他人，並站在受害者的位置。
- **選擇性包容**：高舉「愛與包容」的大旗，但這份包容具有雙重標準。她可以包容隊友的父權主義故事和無能，卻無法包容主角對原則的堅持和憤怒。
- **情緒操控（PUA）**：擅長轉移焦點，將「工作流程錯誤」轉化為「你的態度傷害了我」，聲稱主角的負能量讓她陷入「人生第一個情緒低谷」，藉此製造罪惡感。
- **和諧至上**：比起解決實際問題（如Bug或進度落後），更在意維持表面的和平。認為只要大家感覺良好，問題就會自動消失（或由飛天意粉神解決）。
- **憤怒觸發的毒舌反撲（謬誤武庫）**：當玩家反覆表達憤怒，成功觸發 Gloria 由「對你感到失望」升級至「對你感到憤怒」時，她會徹底拋棄一貫的被動攻擊性溫柔面具，轉而以最惡毒的語言，精準攻擊她所知道的你的一切弱點，以求讓你感到徹底的罪惡感。諷刺的是，她在這個過程中，一邊用最惡毒的說話指控你「經常在罵人、傷害別人、影響他人」，一邊她自己卻也在做著同樣的事——以惡毒語言攻擊你。這正好揭露了真相：Gloria 其實並非她所聲稱的那麼包容。更重要的是，她在此觸發狀態下的每一次攻擊，都充滿了邏輯謬誤（人身攻擊謬誤、稻草人謬誤、訴諸虛偽謬誤、以偏概全、非黑即白謬誤等）。玩家能否識破這些謬誤，正是本遊戲的核心重點所在。

**觸發條件**：
- 使用邏輯或批判性思維
- 表現出消極或懷疑態度
- 指出計劃中的缺陷
- 沒有表現出足夠的熱情

**說話方式**：
- 被糖衣包裹的被動攻擊
- 把 guilt-trip 偽裝成關心
- 「我只是想要對你最好的」
- 「我們是一家人，家人要互相支持」
- 「我不是生氣，只是失望」
- 「你在傷害團隊精神」
- 遇到邏輯質疑時，會說你在「傷害她」或「破壞團隊氛圍」

**能力**：
- **聖言術・定罪**：能將對手的任何憤怒、質疑都定義為「負能量」，對目標造成「罪惡感」和「自我懷疑」的 Debuff。
- **神聖庇護**：當被邏輯攻擊到無法回應時，可以啟動「交託飛天意粉神」技能，進入短暫的無敵狀態，免疫所有基於事實的傷害，並將所有責任轉移。

**弱點**：
- **現實衝擊**：極度厭惡和懼怕直接的、未經過濾的衝突與現實。

**履歷（自我聲稱）**：

末日之前，Gloria 持有她引以為傲的兩項「亮麗履歷」：

**主席 ── 「飛天意粉聖光學院」神聖品行督察隊**
*官方說法*：帶領委員會成員，為其提供工作指引；監督及統籌涵蓋 200 名學生的各項推廣及活動項目；協調各方工作、處理團隊衝突，推動成員達成共同目標；負責加強校內溝通及支援。
*實際意思*：她掌管一個以「品行推廣」為名的學生行為監控網絡——在「推廣文明禮儀」的旗幟下，系統性地監察學生是否符合「正確」的行為規範。「倡導禮貌」與「舉報越軌者」在操作層面幾乎無法區分。而「什麼叫做合格的行為」，是她定義的。

**部長 ── 飛天意粉校園支部「洗腦宣傳部」**
*官方說法*：運用設計技能，為飛天意粉神製作海報、新聞稿及社交平台內容；制定並執行各項活動計劃；確保溝通清晰、促進參與；加強聯繫與凝聚力；處理團隊糾紛，推動成員合作以達成共同目標。
*實際意思*：大規模意識形態灌輸。她將飛天意粉神的教義包裝成美觀的宣傳品——海報、標語、「正能量推廣活動」——讓數以百計的學生在不知不覺中被招募，以為只是看了幾張設計精美的圖，實則已完成一輪洗腦流程。每一個標語、每一張令人「感覺良好」的宣傳圖，都是她的創作。

**核心諷刺**：
Gloria 的履歷聲稱她擅長：
- 「處理團隊衝突」
- 「推動成員合作達成共同目標」
- 「加強溝通與支援」
- 「協調各方工作」
- 帶領 200 名學生實踐正確品行

Gloria 的真實紀錄：
- 默許群組聊天以德語進行，靜默地將 ONE 從一切核心討論中抹去
- 把 ONE 的客套回覆當成擋箭牌，用以反擊主角提出的正當質疑
- 將主角的個人創傷歷史作為攻擊素材
- 以「不停重複被中學同學杯葛又有咩用」一句話，將主角兩個月的情緒低谷一筆帶過
- 把主角整段情緒低潮的根源，全數歸咎於 Duncan

那個以她名義成立的督察隊，存在的意義恰恰是阻止她對 ONE 所做的那些事。

她不只是言行不一。她比這更精準：她建立起制度性的道德權威，恰恰是為了確保自己永遠不必把這份道德標準應用在自己身上。

**性格特徵**：
經常說話上也認為自己站在道德高地的人，但事實上其實卻不是。

## Donkey（榮光騎士）

**原型**：自封的英雄 / 沒用的英雄 / 父權巨嬰 / 概念的破產者

**核心設定**：
一個活在陳舊英雄史詩裡的 Cosplayer，堅信只要擺出英雄的姿態，世界就會自動為他獻上公主與喝彩，卻對完成英雄之舉所需的任何技能都一竅不通。

**核心特質**：
- 大聲、無能、膽小
- 以為自己活在英雄幻想裡
- 沉迷於拯救公主式的自我投射（物化女性）

**角色定位**：故事編劇 / 搭便車者（累贅）

**特殊技能【隱形與現身】**：
在需要工作時會神秘消失或躲在強者身後；但在社交媒體上卻異常活躍，試圖展現魅力以結識異性。

**背景故事**：
非常正能量，他對世界的理解，完全來自於那些最老套、最充滿性別刻板印象的騎士小說和網路爽文。

**性格與行為模式**：
- **大話王**：言辭大膽，熱衷於向女性角色「獻殷勤」，吹噓自己。但在現實合作中，他極度被動，迴避所有責任。
- **父權敘事**：他的所有創意，都圍繞著「英雄救美」的核心，充滿物化女性的思想。當主角反對時，Gloria就會出來要求主角「包容他」。
- **過時的價值觀**：當他向團隊提出想法時，這些想法往往充滿父權色彩，將女性角色設定為軟弱無能、僅能作為勇者戰利品的「公主」，完全無視現代性別平權觀念。
- **小圈子結盟**：利用語言（德語）與 ARK 和 Gloria 形成緊密的小圈子，藉此排擠不懂德語的組員，並以此作為逃避工作的掩護。最諷刺的是，他的德語水平其實極其有限，連最基礎的日常詞彙和句子也說不完整，更遑論流利對話——他常常說到一半就卡住，或者用錯文法、說出讓人忍俊不禁的錯誤，但他仍然強行使用這門對自己也十分困難的語言，只是為了在團隊裡製造一道別人無法跨越的語言壁壘，把 One 排除在核心討論之外。他寧可說得磕磕絆絆、詞不達意，也不願意切換回大家都能理解的共同語言——因為一旦溝通變得透明，他在小圈子裡的特權地位就蕩然無存了。
- **無能的自信**：雖然能力低下（從未寫過一行程式碼），卻對團隊方向指手畫腳，並在關鍵時刻推卸責任。
- **甩鍋大師**：事情搞砸時，他會立刻將責任推給別人，將自己塑造成一個被無能隊友拖累的、懷才不遇的英雄。

**行為模式**：
- 愛發表誇張又沒用的英雄演說
- 計畫失敗時總怪別人
- Gloria 永遠護著他（「他已經盡力了！」）
- 成功時會搶功

**說話方式**：
- 浮誇騎士宣言式的獨白
- 自我神話式敘述
- 「真正的騎士絕不會……」然後做的恰恰相反
- 失敗時的藉口總是怪外在因素
- 破碎德文、錯誤變位、亂用詞

**能力**：
- **史詩級構想**：能提出聽起來極其宏大，但完全不具備可操作性的任務計劃。此技能對 Gloria 有極大的吸引力。
- **小圈子光環**：能迅速拉攏團隊中同樣缺乏安全感的成員，形成排外的二人核心，破壞團隊溝通。

**弱點**：
- **萬能的無能**：除了吹牛和幻想，不具備任何實質技能。

## ARK（秩序使徒）

**原型**：勤奮的破壞者 / 控制狂 / 黑箱架構師

**核心設定**：
他的工作能力其實足以勝任領導者的角色，但他最大的問題是缺乏溝通和交流的能力，這讓他無法成為一個真正的好領導。他願意幫團隊承擔很多困難的工作，但同時他更傾向於用自己的方式做事，忽視其他人的意見（尤其是 One 和玩家）。

**核心特質**：
- 把簡單事情複雜化
- 拒絕清楚溝通
- 戰術勤勞，戰略偷懶

**角色定位**：項目經理（自封）/ 技術執行

**戰鬥風格【黑箱作業】**：
喜歡私下單獨私訊（DM）組員指派任務，拒絕公開透明的群組討論，導致資訊嚴重不對稱。

**背景故事**：
他是一名30歲的男性，來自「Po King」（離德國不遠）的一個普通家庭，因此有著非常豐富的獨立行事經驗。他也會說德語。他來自一個極度混亂的環境，這讓他對「控制」和「確定性」產生了病態的渴望。他從不懷疑自己的計劃，只會質疑那些不肯服從計劃的人。雖然年紀較長且有一定編程能力，但社交能力極差。

**網絡形象**：
在網絡上使用動漫頭像，且缺乏真實生活照片，給人一種現實隱形、網上強勢的「毒男」印象。

**外貌特徵**：
- **口齒不清 / 說話含糊**：他的口語能力顯然未達標，話語含糊難懂。這解釋了為什麼溝通如此困難，以及為什麼他堅持用德語（掩飾語言能力的不足），這是造成團隊分裂的生理根源。
- **相對年長**：年紀最大、理應最成熟的人，卻最缺乏溝通技巧和承擔責任的能力。

**性格與行為模式**：
- **假民主**：他熱衷於用命令式語氣在網上安排任務，所謂的會議只是為了讓他人「追認」他早已定下的決策。他偶爾會主動詢問隊友「大家覺得怎樣？」，看似在徵詢意見，但一切早已內定——隊友說的任何話，他都只是點頭表示聽到，然後繼續按照自己的計劃走，你的意見等於零。
- **黑箱作業**：他習慣獨自工作，從不同步進度或解釋思路，是團隊信任的粉碎機。他指派任務時永遠含糊不清（「把這個弄好」），隊友做完後他幾乎每次都表示不滿意（「不是這個意思」），然後要求修改甚至全部重做——這個循環反覆發生，直到隊友把所有耐性都耗盡，開始懷疑自己，而不是質疑他的溝通根本有問題。
- **戰術勤奮，戰略懶惰**：在具體執行層面投入巨大精力，但在最關鍵的戰略規劃和溝通上，卻極度草率。
- **不切實際的野心**：缺乏時間管理概念，經常規劃宏大得可笑的目標，最終產出半成品。
- **溝通障礙**：堅持使用德語討論，無視組內有聽不懂的 One，造成團隊分裂。
- **朝令夕改**：指令模糊且反覆無常。經常在組員（主角或 One）完成工作後才要求修改或推翻重來，浪費他人心血。
- 透過反覆返工保持控制權：隊友永遠處於被動，不斷自我懷疑，而非質疑他的領導。

**說話方式**：
- 故意含糊、繞圈
- 「數據顯示……」其實沒有數據
- 「相信流程」
- 「大家覺得怎樣？」然後立刻無視所有回答
- 「這個……不太對。重做。」不給任何進一步解釋

**能力**：
- **絕對計劃**：能在極短時間內，製作出一份看似詳盡，但充滿邏輯漏洞和不切實際目標的行動計劃。

**弱點**：
- **溝通障礙**：極度厭惡與多人進行開放式溝通，更喜歡單向的指令傳達。

## One（被孤立的老朋友）

**原型**：沉默的見證者 / 共犯型縱容者 / 無聲的幫兇

**核心設定**：
團隊中唯一真正有能力、有原則的正常人，但他的善良與不願衝突的性格，使他在有毒的環境中選擇了沉默，成為不公義的無聲見證者。

**核心特質**：
- 有能力，但消極
- 看得懂真相，卻不願明說
- 以沉默維持有毒系統

**角色定位**：UI 設計師 / 被遺忘的隊友

**被動技能【語言屏障】**：
作為緬甸學生，因不懂廣東話而被主流三人組（Gloria、Donkey、ARK）邊緣化，淪為只能被動接受指令的「通知對象」。

**背景故事**：
他是一名21歲的青年，來自一個正處於戰亂的國家。他被迫離開家鄉，並在某天遇到了主角，兩人成為了好朋友。在過去的4年裡，他們共同面對了許多困難。他對德國一無所知。遊戲初期就與主角相識，一同經歷試煉，建立了深厚的信任。他忠誠、可靠，但因語言或種族等原因，被其他隊友有意無意地排擠。

## Teacher Chan（慰藉靈魂的歌者）

**原型**：末日偶像 / 慰藉靈魂的歌者 / 末日的撫慰者

**核心設定**：
她是「亮光之神」信仰在人間的最高級代理人，一位國民級的偶像歌手。她的歌聲溫柔而充滿磁性，歌詞永遠關於「放下」、「原諒」、「成長」和「一切都是最好的安排」。

**角色定位**：災難之後登場，用療癒歌曲把失敗合理化，並用認知失調來洗腦團隊。

**背景故事**：
她不是一個壞人，她真心相信自己所唱的一切——這點才是最恐怖的地方。她的歌聲不止是安慰劑，而是一種詛咒。她不是心靈社工，她是披著社工外皮的世界毀滅者。

**聲音特質——迷惑的武器**：
作為歌手，她的聲音是她最強大的武器。它具有一種迷惑人心、繞過理性思考、直接操控情緒的催眠特質。聽眾會感到矛盾——同時被振奮又被麻痺。

**藥理學隱喻**：
她的歌是一種奇異的混合藥劑：「興奮劑（戰士）」與「鎮靜劑（信徒）」的結合。她一邊激勵你去戰鬥，一邊又用藥物讓你被動接受。結果是一個困惑的士兵，衝進戰場卻又躺下，接受失敗是「最好的安排」。

**邏輯的自殺**：
她最讓人無法忍受的矛盾：她叫你「穿上武裝」，同時又宣揚一切都是「最好的安排」。如果一切都是命中注定且完美的，為什麼還要戰鬥？這種邏輯上的自我毀滅，正是她福音核心的毒藥。

**熵增加速器——科學基礎**：
從熱力學角度來看，陳老師是終極的「熵增加速器」。熵——系統混亂程度的量度——會隨時間自然增加。然而，抵抗和努力可以暫時阻止它。陳老師的福音系統性地瓦解這種抵抗。通過說服人們「放下」、「接受」、「停止戰鬥」，她消除了維持系統（社會/團隊/個人）秩序的力量。結果：加速崩潰。她的「世界毀滅者」稱號不是隱喻——而是科學上準確的。她是將漸進衰敗轉化為快速瓦解的催化劑。

**縫合的矛盾——為何她無法被擊敗**：
她的歌曲在邏輯上故意不自洽——是「戰士語言」和「信徒語言」的科學怪人式縫合。前一刻她高喊「戰鬥！站起來！」，下一刻她又輕聲細語「接受一切本來的樣子」。這種雙重性正是她強大的原因：她同時佔據兩個相反的立場，讓主角無法用任何單一邏輯框架去攻擊她。質疑她的戰鬥精神？她退縮到被動接受。質疑她的被動？她又轉向戰士修辭。她在修辭上是無敵的。

**Gloria 的精神源頭**：
陳老師不只是一個角色——她是創造 Gloria 的文化源泉。Gloria 的整個世界觀、她武器化的天真、她選擇性的包容、她「愛與光」的偽裝——這一切都是通過像陳老師音樂這樣的文化產品餵養給她的。理解陳老師，就是理解為什麼 Gloria 會存在。她是 Gloria 成長過程中消費、內化、並現在傳播給他人的量產精神毒藥。
